# Bridge v2 Preview Evidence Index

The `2026-07/` directory contains dated preview closeouts, canaries, long-run
notes, early architecture audits, and source-binding evidence moved from
`STS2MCP/docs/bridge-v2/` during the 2026-07-22 consolidation.

These files remain useful for exact environment provenance and design history.
They are not the current protocol, coverage, compatibility, or permission
authority. Current Bridge documentation is in
[`../../STS2MCP/docs/bridge-v2/`](../../STS2MCP/docs/bridge-v2/).

The archive retains three broad classes:

- `PREVIEW_*`: bounded canary/qualification and lifecycle records;
- dated `*_AUDIT_*` and `*_CLOSEOUT_*`: historical analysis and migration
  snapshots;
- baseline/upstream/smoke records: reference material tied to their original
  exact build or observed runtime.

Historical `qualified` language applies only to the recorded game build,
Modset, Bridge artifact, MVID, runtime instance, and operation scope. It does
not transfer to the current checkout.
