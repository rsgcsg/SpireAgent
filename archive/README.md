# Historical Archive

This directory preserves retired code, plans, source audits, preview closeouts,
and evidence. It is intentionally outside the current authority chain.

- `original-spireagent/` preserves the original root runtime and P8--P15
  documentation tree.
- `bridge-v2-previews/` preserves dated Bridge v2 previews, closeouts, and
  historical evidence.

Archived material may be cited for provenance but must not grant current
runtime compatibility, action permission, feature support, or roadmap status.
See [`../docs/current/DOCUMENT_MAP.md`](../docs/current/DOCUMENT_MAP.md).
